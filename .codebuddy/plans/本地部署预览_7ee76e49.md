## 用户需求

在本地启动开发服务器，查看项目效果，并在启动后自动打开浏览器访问。

## 核心功能

- 将 vite.config.ts 的 base 从 '/historyStory/' 改回 '/'（本地不需要路径前缀）
- 启动 Vite 开发服务器（npm run dev）
- 在浏览器中打开本地访问链接

## 技术方案

- 项目技术栈：React + Vite + TypeScript + Tailwind CSS
- 本地启动方式：`vite dev`（即 npm run dev），默认端口 5173
- 需要将 `base: '/historyStory/'` 改回 `base: '/'`，否则本地访问需要加 `/historyStory/` 前缀
- PowerShell 执行策略问题需用 `powershell -ExecutionPolicy Bypass` 绕过

## 实施步骤

1. 修改 vite.config.ts：去掉 `base: '/historyStory/'`
2. 用 `powershell -ExecutionPolicy Bypass -Command "npm run dev"` 启动开发服务器
3. 在浏览器打开 `http://localhost:5173`
